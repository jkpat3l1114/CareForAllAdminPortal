# CareForAll Admin Backend Portal

An interactive prototype of the CareForAll Super Admin command center — a single self-contained `index.html` (mock data, no backend).

## View
Open the GitHub Pages link, or open `index.html` in any browser.

## Pages
Community Impact Dashboard & Resources · Mapping · Project Hub · Mentorship · Chapter Hub · Service Hours · Forms & Events · Profile & Settings.

## Remix
Fork this repo and edit `index.html` — everything (styles, mock data, logic) lives in that one file. Use the "View as" role switcher (top right) to preview each admin role.

_Prototype only. All names and data are fictional._
